# codex / gpt-image2 prompt 模板

> 重要：不指定具体模型名（让 codex 用其默认模型）；**显式提醒** codex 使用 **gpt-image2** 进行图像生成（gpt-image2 是图像能力，不是模型名）。
> 全部 prompt 与图内文字一律简体中文。

## 调用方式

通过已安装的 codex 插件调起，例如：
```
codex exec "{下面拼装好的 prompt 全文}"
```
若用户的 codex 插件提供了 slash 命令（如 `/codex:run`），同样可用，关键是把 prompt 完整传入。

## (A) 概念示意图 prompt 模板

```
请使用 gpt-image2 进行图像生成。模型用你默认的即可。

绘制内容：{从 spec.md 抽取的核心描述，含主题、模块清单、连接关系}。

风格要求：
- 学术风、扁平、可矢量化（线稿 + 适度填色）；
- 主色：{spec 给出的主色 hex}；辅色：{辅色 hex}；
- 白色背景，无投影；
- 图内**全部文字必须使用简体中文**，包含模块名、箭头标签、图例；
- 不要出现英文术语（除非属于公认缩写如 CNN、MLP，且需在括号内附中文）；
- 字体清晰、字号统一。

技术要求：
- 输出分辨率 ≥ 1600×1200；
- PNG 格式；
- 输出文件保存到：{PROJECT_ROOT}/docs/paper/figures/{name}.png

如不能直接写文件，请把 base64 或图像 URL 输出，由我后续保存。
```

## (B) 数据图表预览 prompt 模板

```
请使用 gpt-image2 生成一张**预览草图**（明示：仅用于审稿，最终将由科学绘图工具重绘）。

数据：
{从 data.md 抽取的数据表，按"类别 → 数值"列出}

图类型：{柱状图 / 折线图 / 热力图 / 箱线图 / ...}
中文标题：{title}
横轴标签：{xlabel}（中文）；纵轴标签：{ylabel}（中文）；图例：{legend 中文}
配色：{2~4 个 hex 色值，符合学术配色}

风格要求：
- 学术、简洁、白底、无 3D；
- 全部文字简体中文；
- 网格线浅灰；
- 输出分辨率 ≥ 1600×1200；
- PNG 保存到：{PROJECT_ROOT}/docs/paper/figures/{name}.preview.png
```

## 提示给用户的话术

预览出图后告知：
> 这是 gpt-image2 预览草图，仅用于布局/配色审阅。最终图请用我下面推荐的科学绘图工具基于真实数据重新绘制，避免 AI 生成图在期刊/查重中被识别为合成图。
