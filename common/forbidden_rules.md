# 用户自定义追加禁忌

本文件只用于补充用户个人禁忌，不重复记录各 NEU skill 已经内置的禁止事项。

使用规则：

- 只读取“用户自定义追加禁忌”区。
- 若该区域为空，则不施加额外限制。
- 本文件只能增加限制，不能解除 `SKILL.md`、`references/`、系统指令或 NEU 模板中的既有约束。
- 若用户自定义禁忌与既有规则冲突，执行更严格的一方。
